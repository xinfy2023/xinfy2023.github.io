# Feiyang Xin Homepage (Static)

本目录是一个纯静态个人主页（HTML/CSS，双语）。

## 入口

- 中文：`index_zh.html`
- English：`index.html`

## 文件说明

- `assets/style.css`: 页面样式
- `1.jpg`: 主页头像（当前使用）
- `assets/avatar.svg`: 头像占位（无照片时可回退使用）
- `files/`: 附件（简历与论文 PDF）

## 本地预览

直接双击打开 `index_zh.html` / `index.html` 即可。

## 发布到 GitHub Pages（推荐）

最简单方式：把本目录内的文件复制到 GitHub Pages 仓库根目录（确保根目录有 `index.html`）。

## 注意（double-blind）

`E-NGBOOST` 当前标注为 under review，PDF 也显示为匿名稿。若该投稿处于严格 double-blind 阶段，请确认公开在个人主页是否符合投稿政策；如需更稳妥，可以暂时去掉 PDF 链接或改成“available upon request”。
